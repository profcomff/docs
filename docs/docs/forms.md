# Вставка форм в документацию

Часто в документацию требуется встроить форму обратной связи или регистрации. Для этого мы используем [Яндекс Формы](https://forms.yandex.ru/cloud/admin/). Для создания форм настоятельно рекомендуется использовать аккаунт my.msu.ru!

## Порядок вставки формы в документацию
1. Создайте и оформите форму
2. Нажмите кнопку "Поделиться" и скопируйте ссылку на форму
3. Конвертируйте в URL, поддерживаемый приложением, через форму ниже

    {% include notitle [Инжектор форм](../_includes/forms/forms-injector.md) %}

4. *(Опцианально)* Если форма может быть использована в нескольких местах документации, создайте под нее файл [в папке `docs/_includes/forms`](https://github.com/profcomff/docs/tree/main/docs/_includes/forms). Иначе можно выполнить только следующий шаг.
5. Вставьте в файле следующий код: `<iframe src="тут ссылка, которую вы получили в генераторе выше" frameborder="0" width="650" height="500"></iframe>`.
6. *(Опцианально)* Если вы создали файл в папке `_includes` в пункте 4, то в конечный файл форма добавляется так: `{% include notitle [Название формы](путь/к/файлу.md) %}`
