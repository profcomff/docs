# Компоненты платформы

В платформу Твой ФФ входят следующие приложения

## Разрабатываемые непосредственно командой разработки
- Приложение Твой ФФ:
  - Бэкэнд:
    - API аутентификации и авторизации: <https://api.profcomff.com/?urls.primaryName=auth>
    - API пользовательских данных: <https://api.profcomff.com/?urls.primaryName=userdata>
    - API ачивок: <https://api.profcomff.com/?urls.primaryName=achievement>
    - API календаря: <https://api.profcomff.com/?urls.primaryName=timetable>
    - API списка сервисов: <https://api.profcomff.com/?urls.primaryName=services>
    - API исследований поведения пользователей: <https://api.profcomff.com/?urls.primaryName=marketing>
  - Фронтэнд: <https://app.profcomff.com>
- Приложение Бесплатный принтер:
  - Бэкэнд: <https://api.profcomff.com/?urls.primaryName=print>
  - Фронтэнд: <https://app.profcomff.com/apps/1>
- Приложение слежения за стабильностью инфраструктуры: <https://api.profcomff.com/?urls.primaryName=pinger>
- Приложение мониторинга социальных сетей: <https://api.profcomff.com/?urls.primaryName=social>
- Сервис брендирования ссылок: <https://to.profcomff.com/>
