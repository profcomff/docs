# Общая информация
На сегодняшний день существуют десятки платформ и операционных систем, сотни и даже тысячи способов разработки под эти платформы. Чтобы не мучиться с разработкой различных приложений на все популярные платформы мы приняли решение использовать современные веб технологии.

Для разработки приложений мы используем [Node.JS](https://nodejs.org/). Большинство приложений написаны с использованием фраемворка [Vue.JS](https://vuejs.org/index.html), который рекомендуется использовать и для новых сервисов. Это позволит унифицировать все сервисы, чтобы их было проще передавать будущему поколению студентов.

Чтобы приложение удобно работало с любого устройства, могло работать оффлайн и поддерживало множество других крутых функций, мы используем технологию [PWA](https://web.dev/progressive-web-apps/). *PWA (Progressive Web App) – это веб-технология, которая трансформирует сайт в приложение. Прямо из браузера его можно поставить на главный экран телефона, и оно будет отправлять push-уведомления и получит доступ к аппаратным средствам гаджета. И все это даже при нестабильном подключении или офлайн.*

Для скорости и удобства разработки мы используем [библиотеку компонентов Vuetify](https://vuetifyjs.com/en/components/all/), реализующую Material Design.

# Разработка
## Среды выполнения
У нас есть 2 среды:
* Production <https://app.profcomff.com/>. Среда для конечных пользователей. В этой среде запрещены devtools и в нее не выкатываем непроверенных изменений.
* Testing <https://app.test.profcomff.com/>. Среда для тестирования и разработки. В ней можно проводить эксперименты не боясь что-то сломать конечным пользователям.

## Локальная разработка
Чтобы быстро влиться в разработку, рекомендуется проделать инструкции из репозитория <https://github.com/profcomff/webapp-ui>
