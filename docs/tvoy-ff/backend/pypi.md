# PyPI

## Что это такое?
`PyPI` — каталог программного обеспечения, написанного на языке программирования Python. Фактически это хранилище открытых библиотек для языка Python. Каждый может скачать и использовать библиотеку или создать и загрузить свою.

В Твой ФФ он используется для хранения библиотек. По состоянию на август 2024 года туда загружено 5 наших компонентов.

## Как использовать его для деплоя компонентов
Токен для доступа в `PyPI` лежит в секретах организации, подробнее можно посмотреть в разделе `CI/CD`

С помощью этого токена осуществляется доступ к уже существующим проектам, а так же его можно использовать для деплоя новых

Чтобы здеплоить новую библиотеку:

1. Разрешите использование токена в новом репозитории
2. Закомитьте новый код и папйплайн, папйплайн можно взять отсюда: <https://github.com/profcomff/auth-lib/blob/main/.github/workflows/build_and_publish.yml>. Структура проекта тоже должна быть похожа.
3. При первом создании тега библиотека задеплоится в наш `PyPI`, доступ по дефолту будет у владельца токена, он будет указан во владельцах пакета


## Как получить доступ к PyPI
1. Создайте аккаунт на `PyPI`
2. Попросите одного из владельцев пакета, нужного вам для управления, выдать вам доступ

## Что можно делать с доступом к пакету
1. Удалять пакет, изымать версии, удалять версии
2. Добавлять соавторов
3. Создавать токен для деплоя _данного пакета_ на `PyPI`. Это отличается от токена аккаунта
