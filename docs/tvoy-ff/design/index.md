# Дизайн приложений Твой ФФ

В разработке дизайна Твой ФФ и микросервисов для него мы придерживаемся стиля Material Design 3 и готовых компонентов приложения от [Vuetify](https://vuetifyjs.com/en/components/all/#containment). Это позволяет не только быстро отрисовывать приложение из готовых компонентов и унифицировать интерфейс приложений, но и заметно ускорить разработку этих интерфейсов в коде.
