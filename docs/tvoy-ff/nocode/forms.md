# Добавление кнопок в меню

Для добавления кнопки в меню, нужно сделать запрос к [Service API](/api/services/index.html). В данный момент, это делается через обращение к разработчикам. Планируется сделать панель администратора, чтобы упростить этот процесс.

Добавляемый элемент должен иметь возможность встраивания черех плавающие фреймы (iframe).

## Добавление форм

Чтобы добавить форму в меню, нужно собрать форму через конструктор [Яндекс Форм](https://forms.yandex.ru/cloud/admin/) и конвертировать ее в URL, поддерживаемый приложением, через форму ниже. Для создания форм настоятельно рекомендуется использовать аккаунт my.msu.ru!
